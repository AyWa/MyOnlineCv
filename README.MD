# Web CV


## Installation

To use this project, your computer needs:

- [NodeJS](https://nodejs.org/en/)
- [Git](https://git-scm.com/)

### Using the CLI

```
npm install

```
### to test on chrome localhost:

```
npm install -g http-server

```
then in the project directory:

```
http-server

```
